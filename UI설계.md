# UI 설계 → 이메일 알림 설계로 대체

> 웹 앱 없이 Google Form + 이메일 알림으로 운영하므로, 기존 UI 설계(페이지 구조/와이어프레임)는 불필요.
> 사용자 접점은 두 가지: (1) Google Form, (2) 이메일 알림.
> 이메일 템플릿 상세 설계는 구현계획_구체.md에 포함.

## 사용자 접점 정리

| 접점 | 도구 | 용도 |
|------|------|------|
| 가입 + 스펙 입력 | Google Form | 프로필 등록 (3섹션: 기본정보/스펙/희망조건) |
| 프로필 수정 | Google Form 수정 링크 | 제출 후 이메일로 수정 링크 자동 발송 |
| 분석 결과 수신 | 이메일 (Gmail) | 매칭된 공고별 분석 리포트 |
| 원문 확인 | 잡알리오 링크 | 이메일 내 원문 바로가기 링크 |
