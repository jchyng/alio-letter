## 기술 스택

| 구분 | 선택 | 비고 |
|------|------|------|
| 프론트엔드 | Cloudflare Pages (정적 HTML) | Pretendard 폰트, 모바일 퍼스트 |
| 백엔드 API | Cloudflare Functions (Workers) | 서버리스, Edge 기반 |
| DB 연결 | Cloudflare Tunnel (`cloudflared`) | 포트 개방 없이 라즈베리파이 MySQL 직접 연결 |
| 데이터베이스 | MySQL 8 (라즈베리파이 Docker) | 관계형 데이터 처리 |
| AI 파싱 | Gemini 1.5 Flash API | 스펙 파싱 + 공고 PDF 추출 + 자격증 점수 계산 |
| 이메일 발송 | Resend API | 확인 메일 + 분석 레포트 |
| 배치 파이프라인 | Python (`pipeline/`) | FLOW 1~4 전체. crontab으로 스케줄링 |
| 공고 스크래퍼 | Python (`pipeline/scraper.py`) | `python main.py --flow collect`로 실행 |
| 스케줄링 | 라즈베리파이 crontab | n8n 불필요 — Python이 직접 처리 |
| 결제 | 베타 기간 무료 → 추후 토스페이먼츠 | 현재 미구현 |

## 시스템 아키텍처

```
[사용자 브라우저]
    │
    ├── 정적 페이지 요청
    │       ↓
    │   [Cloudflare Pages]
    │   /  /register  /login  /profile/:token
    │
    └── API 요청
            ↓
        [Cloudflare Functions (Edge)]
            │
            ├── Gemini API 호출 (스펙 파싱)
            ├── Resend API 호출 (가입 확인 메일)
            └── Cloudflare Tunnel → MySQL 쿼리

─────────────────────────────────────────────

[라즈베리파이 (Home Network)]
    │
    ├── cloudflared (Tunnel Daemon)
    │     Cloudflare와 아웃바운드 터널 유지
    │
    ├── MySQL 8 (Docker)
    │     사용자 데이터 + 공고 데이터 + 분석 결과
    │
    └── Python Pipeline (crontab)
          pipeline/main.py --flow collect  [매일 06:00]
            → scraper.py   잡알리오 크롤링 → postings 저장
            → parser.py    Gemini PDF 파싱 → posting_tracks 저장
            → matcher.py   사용자 매칭 쿼리
            → analyzer.py  자격 체크 + 점수 계산 → user_posting_analyses 저장

          pipeline/main.py --flow email    [매일 07:00]
            → mailer.py    Resend API 이메일 발송

          pipeline/main.py --flow expire   [매일 00:00]
            → 마감 공고 status 업데이트
```

---

## 역할 분리 원칙

| 처리 유형 | 담당 | 이유 |
|-----------|------|------|
| 실시간 (사용자 요청) | Cloudflare Functions | 빠른 응답 필요 |
| 배치 (공고 수집·발송) | Python pipeline (라즈베리파이) | 장시간 실행, CF Functions 제한 회피, 단일 언어 |
| AI 파싱 (스펙) | CF Functions → Gemini | 사용자 요청 흐름 내 동기 처리 |
| AI 파싱 (공고 PDF) | parser.py → Gemini | 배치 흐름 내 처리 |
| 자격 체크·점수 계산 | analyzer.py (Python 직접 계산) | 정확성·재현성 보장 (LLM 미사용, 자격증 점수만 Gemini) |

## 보안 연결 구조 (Cloudflare Tunnel)

```
[Cloudflare Functions]
    │  mysql2 라이브러리
    │  접속 주소: db.internal.example.com (Tunnel 도메인)
    ▼
[Cloudflare Zero Trust]
    │  Worker IP만 접근 허용 (Access Policy)
    ▼
[cloudflared on 라즈베리파이]
    │  아웃바운드 전용 터널
    ▼
[MySQL Docker :3306]
```

> Python pipeline은 라즈베리파이 내부에서 직접 `localhost:3306`으로 MySQL에 접근 (Tunnel 불필요)

---

## 실행 스케줄 (crontab)

| 명령 | 시각 | 내용 |
|------|------|------|
| `--flow collect` | 매일 06:00 | scraper → parser → matcher → analyzer |
| `--flow email` | 매일 07:00 | mailer (Resend API 발송) |
| `--flow expire` | 매일 00:00 | 마감 공고 status 업데이트 |

## 라즈베리파이 Docker 서비스

```
- mysql8:      MySQL 8 (포트 3306, 내부 전용)
- cloudflared: Cloudflare Tunnel 데몬 (아웃바운드 전용)
```

> n8n 컨테이너 없음.
