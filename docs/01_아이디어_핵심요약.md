## 해결하고자 하는 문제

1. 공기업 취준생들이 공부에만 집중해도 모자란데 채용공고까지 매일 확인해야함.

<br>

## 어떻게 해결해줄 것인가?

1. 채용공고는 자동으로 시스템이 읽고, 요약해서 긴 내용의 문서를 읽을 필요없게 해줌.
2. 사용자의 현재 스펙 정보를 토대로 지원할 수 있는 회사를 추려서 간소화

<br>

## 취준생들이 채용 공고에서 보는 정보는?

### 1. 내가 가고싶은 회사인가?

- 지역
- 직무
- 기업명
- 연봉

### 2. 내가 지원할 수 있나?

- 자격 요건(학력, 채용구분, 어학성적, 자격증 등)

### 3. 합격할 수 있나?

- 가점 기준
- 채용 구분
- 채용 인원
- NCS + 전공 출제 범위

### 4. 지원하겠다고 마음 먹고 나서 필요한 정보

- 채용 절차
- 지원 방법
- 마감 기한

<br>

## 사용자에게 무엇을 제공해줄 것인가?

1. 사용자가 원하는 조건(기업, 지역, 직무, 연봉)에 해당하는 기업만 필터링
2. 사용자의 스펙 정보를 토대로 자격요건 검토 => 적부 체크 시각적으로 보여줌
3. 사용자의 스펙 정보를 토대로 가점 평가 => ex.72/100점 + 각 평가 기준과 계산을 보여줌으로써 신뢰도 up
4. 지원 바로가기 + 잡알리오 상세페이지 바로가기로 더 많은 정보 보기

<br>

## 전달 방식

- ~~웹 앱 (Flask + PWA)~~ → **이메일 알림**으로 변경
- Google Form으로 가입/스펙 입력/수정
- 매칭 결과를 이메일로 자동 발송 (요약 + 자격체크 + 가점분석 + 원문링크)
- 추후 사용자 증가 시 카카오 알림톡 추가

<br>

## 기술 인프라

- ~~Flask + SQLite + Oracle EC2~~ → **n8n (라즈베리파이) + Google 생태계 + SQLite**
- 사용자 관리: Google Form → Google Sheets
- 자동화: n8n 워크플로우 (스크래핑 → PDF 파싱 → 매칭 → 이메일 발송)
- PDF 파싱: Gemini API

<br>

## 돈은 어떻게 벌 것인가?

- 구독제로 매달 1900원 or 2900원 결제
- 결제: 계좌이체 → 수동 확인 → Google Sheets에서 활성화

<br>

  > 합격 스펙 데이터를 토대로 현재 사용자의 서류 합격률 보여주기
