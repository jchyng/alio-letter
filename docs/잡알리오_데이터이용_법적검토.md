# 잡알리오 데이터 이용 법적 검토

## 결론: ✅ 이용 가능

## 근거

1. **저작권법 제24조의2** — 잡알리오가 저작재산권 전부를 보유한 공공저작물은 별도 허락 없이 자유이용 가능
2. **공공데이터법 제1조, 제3조** — 영리 목적 포함 자유 활용 보장

## 조건

- 출처 표시 필수 (기관명 + URL)
- 공공누리(KOGL) 유형 조건 준수

## 제외 대상

- 제3자 권리 포함 정보: 개별 기관 로고, 특정 저작물 등 (공공데이터법 제17조, 정보공개법 제9조)
- 개인정보 포함 데이터: 개인정보보호법 별도 적용
